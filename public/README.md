Aplikasi manajemen laporan penjualan harian untuk toko, dibuat dengan HTML, CSS, dan JavaScript (Firebase).

## Fitur Utama

- **Input Data Penjualan**: Mencatat penjualan shift 1 & 2, struk, dan otomatis menghitung total.
- **Analisis Pertumbuhan**: Menampilkan pertumbuhan (growth) SPD, STD, dan APC dibandingkan bulan sebelumnya.
- **Manajemen Akses**: Sistem role (Super Admin, Admin, Staff) dengan hak akses berbeda.
- **Audit Log**: Mencatat aktivitas pengguna untuk keamanan.
- **Export/Import**: Backup dan restore data penjualan dengan mudah.
- **Laporan Otomatis**: Generate teks laporan harian siap salin untuk WhatsApp/Telegram.
- **Mode Gelap/Terang**: Dukungan tema visual yang nyaman.

## Struktur Kode

- `index.html`: Struktur utama aplikasi.
- `styles.css`: Styling dan tema.
- `app.js`: Logika aplikasi, perhitungan, dan integrasi Firebase.

## Cara Menggunakan

1. Buka `index.html` di browser.
2. Login menggunakan akun Google (jika fitur auth diaktifkan).
3. Input data penjualan harian di tabel.
4. Gunakan tombol "Copy Laporan" untuk menyalin ringkasan harian.
